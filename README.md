# YBY Compiler - Free Online Code Editor for Python, C, C++, and Java

**YBY Compiler** is a modern, browser-based integrated development environment (IDE) designed for students, educators, and developers who want to write, compile, and execute code instantly without installing anything.

## 🌟 Features

- **Multi-Language Support**: Write Python, C, C++, and Java code in a single platform
- **Browser-Based**: No installation required—start coding immediately
- **Python Data Science**: Built-in support for NumPy, pandas, Matplotlib, seaborn, and scikit-learn
- **File Uploads**: Upload CSV and data files for analysis directly in Python
- **Chart Visualization**: View Matplotlib and seaborn charts inline
- **Beginner-Friendly Error Messages**: Technical errors translated into plain English
- **Code Examples**: Pre-loaded templates to jumpstart learning
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## 🚀 Getting Started

1. **Visit the Compiler**: Open [YBY Compiler](https://ybycompiler.netlify.app/) in your browser
2. **Choose Your Language**: Select Python, C, C++, or Java from the dropdown
3. **Write Code**: Use the editor to write your program
4. **Run It**: Click "Run Code" to execute instantly
5. **See Results**: View output, charts, and helpful error explanations
